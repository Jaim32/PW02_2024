This website

El propósito de este trabajo fue crear un tema integral para un sitio web personal, abarcando la maquetación, tipografía, colores, fondos, formularios y navegación, siguiendo buenas prácticas de desarrollo web, como la accesibilidad y la compatibilidad entre dispositivos y navegadores.


Requisitos y implemtaciones

  • Estructura y Maquetación:

    Diseño adaptable y responsive, optimizado para diferentes tamaños de pantalla. Márgenes, rellenos y alineaciones definidos para las distintas secciones. Implementación de un sistema de rejilla para organizar el contenido.


  • Tipografía:

    Aplicación de fuentes adecuadas para títulos, subtítulos y texto de cuerpo. Personalización del tamaño, peso y espaciado de las letras para asegurar una buena legibilidad.


  • Colores y Fondos:

    Esquema de colores primario, secundario y de acento para una mejor estética. Uso de fondos con colores planos, degradados e imágenes.


  • Formularios:

    Estilos aplicados a los campos de entrada, áreas de texto y botones. Estados de interacción como hover, focus y active para mejorar la experiencia de usuario.


  • Navegación:

    Diseño de barras de navegación, menús y dropdowns con accesibilidad y responsividad. Estilos aplicados a enlaces y botones, incluyendo estados de hover y active.

  • Accesibilidad:

    Asegurar que todos los elementos cumplan con los estándares de accesibilidad, como el contraste de color y los estados de foco.
